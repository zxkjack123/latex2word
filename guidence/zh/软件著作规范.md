# 软件著作规范（登记与文档）

适用范围：聚变工程/核工程领域科研软件的登记与发布。

文档类型
- 软件说明文档：功能、模块结构、环境依赖、使用方法、接口说明、许可与引用
- 用户手册：安装、配置、示例、常见问题
- API 参考：从源码注释自动生成（Sphinx/Doxygen 等）
- V&V 报告：测试、基准、对照参考
- 版本与发布说明：语义化版本
- 许可证与引用：LICENSE、CITATION.cff、AUTHORS

登记（摘要）
- 中国软件著作权登记：按中国版权保护中心/国家版权局最新指南准备材料（权属证明、说明文档、源代码节选页数与格式等）。具体格式、页数及保密处理按官方最新要求执行。

最佳实践
- 制品版本化并归档（如 Zenodo DOI）；提供容器/环境锁定文件以确保可复现。
- 数据集/核数据库引用清晰，遵循许可。

检查清单
- [ ] 软件说明完整，API 文档构建无错误
- [ ] 示例与测试数据可运行复现
- [ ] 许可与引用文件齐备，版本已打标签
- [ ] 登记材料按最新官方要求准备
