# 专利规范（CN/EN 对照要点）

适用范围：聚变工程与核技术相关发明、实用新型等。

核心依据
- WIPO PCT 申请人指南（国际阶段）
- 中国国家知识产权局（CNIPA）《专利审查指南》《专利法实施细则》
- USPTO MPEP（美国）、EPO Guidelines（欧洲）（如涉对应国）

说明书结构（常见）
- 题名
- 技术领域
- 背景技术
- 发明内容（至少一个技术问题与技术方案、效果）
- 附图简要说明
- 具体实施方式（可选实施例、实验/仿真数据、附图对应标号）
- 权利要求书
- 摘要

撰写要点
- 权利要求：独立权利要求覆盖核心构思，从属权利要求逐步限定；一条权利要求一句话，具备先前引述（“所述”）。
- 术语与单位：以 SI 为主；避免商标名；术语在“具体实施方式”中明确。
- 充分公开：能使本领域技术人员实现；给出可选范围与优选值，但避免不必要过窄限定。
- 附图：与说明书标号一致、清晰可读；必要时提供流程图/结构示意图。

合规与保密
- 注意涉密与出口管制；在公开前完成内审与合规评估。

检查清单
- [ ] 技术问题、技术方案、技术效果三要素齐全
- [ ] 权利要求清晰、层级合理、术语前后一致
- [ ] 实施例可实施，附图标注一致
- [ ] 单位与核领域记号规范（见 `../common/units_nuclear.md`）
- [ ] 引用现有技术合规
