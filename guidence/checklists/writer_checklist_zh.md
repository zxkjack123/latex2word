# 写作检查清单（聚变/核）

- [ ] 分级编号符合 ISO 2145；附录 A、B…
- [ ] 以 SI 为主；核领域非 SI（eV、b）使用规范
- [ ] 变量斜体，函数/单位符号正体
- [ ] 数值—单位空格与前缀正确
- [ ] 公式居中编号并引用
- [ ] 图：坐标+单位；位图≥300 dpi，矢量优先；题注简洁
- [ ] 表：三线表；单位置于表头
- [ ] 核素/反应记号正确（²³⁵U，(n,γ)）
- [ ] 不确定度与有效数字一致（GUM 风格）
- [ ] 参考文献：GB/T 7714（中文）或 IEEE（英文）；含 DOI
- [ ] 数据集/软件以 DOI 与版本引用
- [ ] 可复现性：代码/数据可用性、版本记录
