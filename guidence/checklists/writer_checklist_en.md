# Writer Checklist (Fusion/Nuclear)

- [ ] Numbering uses ISO 2145; appendices A, B…
- [ ] SI primary; nuclear non‑SI (eV, b) used appropriately
- [ ] Variables italic; functions, unit symbols upright
- [ ] Value–unit spacing and prefixes correct
- [ ] Equations centered, numbered, and referenced
- [ ] Figures: axis + unit; ≥300 dpi (bitmap), vector preferred; caption concise
- [ ] Tables: three-line style; units in headers
- [ ] Nuclide and reaction notation correct (²³⁵U, (n,γ))
- [ ] Uncertainty and significant figures consistent (GUM style)
- [ ] References: IEEE (EN) or GB/T 7714 (ZH); DOIs included
- [ ] Datasets/software cited with DOI and version
- [ ] Reproducibility: code/data availability, versions recorded
